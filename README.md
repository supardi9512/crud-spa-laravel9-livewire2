## CRUD SPA Using Laravel 9 And Livewire 2

Simple Create, Read, Update, and Delete Single Page Application using Laravel 9 and Livewire 2

### Tech Stack

- Laravel 9
- Bootstrap 5
- Livewire 2
- Turbolinks
- MySQL

### Features

- Read Data
- Create Data (Inc. Validation)
- Update Data (Inc. Validation)
- Delete Data 
- Search Data
